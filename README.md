# 🎓 Student Management System

A Python-based Student Management System that allows users to manage student records using a CSV file.

## 🚀 Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Input Validation
- CSV Database
- Object-Oriented Programming

## 🛠 Technologies Used

- Python
- CSV Module
- OOP
- File Handling

## 📂 Project Structure

```
Student-Management-System/
│
├── main.py
├── student.py
├── database.py
├── utils.py
├── students.csv
├── README.md
└── requirements.txt
```

## ▶️ How to Run

```bash
python main.py
```
